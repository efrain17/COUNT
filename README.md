### ¿Cómo me instalo? ###
* Instalacion de mocha para tdd
* npm install mocha -g
* Instalación de dependecias en el Front-End
* cd /app/ 
* npm install
* Instalación de dependecias de Back-End
* cd /server/
* npm install
